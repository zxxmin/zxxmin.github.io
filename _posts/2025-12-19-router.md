# 들어가며
어떤 프로젝트를 진행하더라도 재사용할 수 있는 기초 틀을 만들고자 base project를 시작했고, 프레임워크에 종속되지 않는 범용성을 고려해 React를 선택하였다.

그런데 React는 Next.js와 달리 폴더 구조 기반의 라우팅을 지원하지 않기 때문에, 메뉴를 추가하거나 메뉴명·URL을 변경할 때마다 개발자가 직접 라우트 설정을 수정해야 하는 번거로움이 있었다.
사용자가 메뉴를 자유롭게 관리할 수 있는 구조에서는 이런 방식이 유지보수 부담으로 이어진다는 문제가 있었다.

이를 해결하기 위해, 사용자가 메뉴에 `menu key` 값으로 연결할 페이지 컴포넌트명을 직접 지정하도록 하는 방식을 떠올렸다. 
메뉴명이나 URL이 바뀌더라도 컴포넌트명을 기준으로 라우팅을 연결하면, 개발자의 개입 없이도 동적으로 라우팅 구조를 유지할 수 있을 것이라는 아이디어에서 이 작업을 시작하게 되었다.

# createBrowserRouter
`React Router v6.4`에서 새로 도입된 라우터 설정 방식.<br/>
라우터 구성을 **객체 형태** 로 선언적으로 만들 수 있게 해주며,이를 BrowserRouter 컴포넌트에 전달하여 라우팅을 설정.

BrowserRouter는 간단한 라우팅만 제공하지만,<br/>
createBrowserRouter는 **데이터 로드와 에러 처리** 등을 함께 지원.

# BrowserRouter 🆚 createBrowserRouter
|                      | BrowserRouter     | createBrowserRouter |
|----------------------|-------------------|----------------------|
| 라우팅 방식 | **컴포넌트 기반** 라우팅 | **객체 기반 선언적** 라우팅 |
| 폼 처리 (action) | ❌ | <form /> 외에도 데이터를 서버와 주고받는 작업 처리 가능 (GET 제외) |
| 에러 처리 (errorElement) | ❌<br/>별도 에러 처리 (*) | **errorElement**로 에러 처리 가능 |
| 데이터 로드 (loader) | ❌ | **loader**를 사용해 라우트 전환 시 미리  로드 가능 |
| [Data API](https://reactrouter.com/6.30.2/routers/picking-a-router#data-apis) 사용 | ❌ | ⭕️ |
| 활용 방법 | 단순 네비게이션과 프로젝트 전환 | 데이터 로드, 폼 제출, 각각의 에러 처리가 필요한 프로젝트 |

# 대표 기능
## root 설정
- 위치가 무조건 최상단 root에 설정이 아니고, App 이나 routes 파일에서도 설정 가능.

### BrowserRouter
- 최상단 root 에서 `BrowserRouter` 설정
```javascript
import { BrowserRouter } from 'react-router-dom'

const root = ReactDOM.createRoot(document.getElementById('root'))

root.render(
    <BrowserRouter>
        <App /> 
    </BrowserRouter>
)
```

### createBrowserRouter
- 최상단 root 에서 `RouterProvider` 설정
```javascript
import { RecoilRoot } from 'recoil';
```

## errorElement : 에러 처리
- 에러가 발생했을 때 처리할 수 있는 errorElement 기능 제공.
- 데이터 로딩 중 에러가 발생하면 미리 정의된 에러 페이지를 표시.
- 각 라우트마다 설정 가능.

### 기본 구조
```javascript
const router = createBrowserRouter([
    {
        path: "/",
        element: <Root />,
        loader: rootLoader,
        children: [
            {
                path: "team",
                element: <Team />,
                loader: teamLoader,
            },
        ],
        errorElement: <ErrorPage />,
    },
]);
```

### 프로젝트 적용
```javascript
import type { RouteObject } from "react-router-dom";
import ErrorPage from "@/pages/auth/ErrorPage";
import { subRouteConfig } from "./subRouteConfig";
import type { MenuListData } from "@/pages/admin/menuManage/types/menuManageTypes";
export const createMenuRoutes = (
    menus: MenuListData[]
): RouteObject[] => {
    return menus
        .filter(menu => menu.component && menu.url)
        .map(menu => ({
            path: menu.url,
            children: subRouteConfig[menu.component!] ?? [
                { index: true, element: <ErrorPage /> },
            ],
            errorElement: <ErrorPage />,
        }));
};
```

## loader : 데이터 로드
- 라우트 전환 시 데이터를 미리 로드.
- 페이지 전환 중 서버에서 데이터를 가져와서 컴포넌트에 전달.
- 컴포넌트에서는 useLoaderData 훅을 통해 값을 사용.

### 기본 구조
```javascript
const userLoader = async ({ params }) => {
    return fetch(`/api/teams/${params.teamId}.json`);
};

const router = createBrowserRouter([
    {
        element: <Teams />,
        path: "teams",
        children: [
            {
                element: <Team />,
                path: ":teamId",
                loader: userLoader,
            },
        ],
    },
]);

function Team() {
    const team = useLoaderData();
    return (
        <div>{team.name}</div>
    )
}
```

### 로딩 상태 관리
```javascript
import {useLoaderData} from "react-router-dom";
import {useState, useEffect} from "react";
const Team = () => {
    const data = useLoaderData();
    const [loading, setLoading] = useState < boolean > (true);
    useEffect(() => {
        if (data) {
            setLoading(false);
        }
    }, [data])
    if (loading) return <div>로딩중...</div>
    return (
        <div>{team.name}</div>
    )
}
```

### createMenuRoutes.tsx
- props로 받은 data(서버에서 받은 메뉴 리스트)를 children에 담겨있는 메뉴들도 routes 변수 요소로 담는 함수
```javascript
import type {MenuListData} from "@/pages/admin/menuManage/types/menuManageTypes.ts";
export const renderRoutes = (menus: MenuListData[]) => {
    let routes: MenuListData[] = [];
    const render = (menus: MenuListData[]) => {
        menus.forEach((menu) => {
            if (menu.component) {
                routes.push(menu);
            }
            if (menu.children && menu.children.length > 0) {
                render(menu.children);
            }
        });
    };
    render(menus);
    return routes;
}
```

### BaseRoutes.tsx
- subRouteMap 의 값들을 Routes 내부에 설정
```javascript
import {Suspense} from 'react';
import { Routes, Route, useLocation } from 'react-router-dom';
import ErrorPage from '@/pages/auth/ErrorPage';
import type {MenuListResponse} from "@/pages/admin/menuManage/types/menuManageTypes.ts";
import {useMenuListHooks} from "@/pages/admin/menuManage/hooks/useMenuListHooks.ts";
import {renderRoutes} from "../../../utils/common/renderRoutes.ts";
const BaseRoutes = () => {
    const location = useLocation();
    const menuList: MenuListResponse = useMenuListHooks();
    const menuRoutes = renderRoutes(menuList.data);
    return (
        <Suspense fallback={<div>loading...</div>}>
            <Routes location={location}>
                {menuRoutes.map((menu) => {
                    const SubRoutes = subRouteMap[menu.component!];
                    return (
                        <Route key={menu.id} path={menu.url as string}>
                            {SubRoutes ?? (
                                <Route index element={<ErrorPage />} />
                            )}
                        </Route>
                    );
                })}
                <Route path='/' element={<div>홈 페이지</div>} />
                <Route path='*' element={<ErrorPage />} />
            </Routes>
        </Suspense>
    );
};
export default BaseRoutes;
```

### createMenuRoutes.tsx
- props로 받은 data(서버에서 받은 메뉴 리스트)를 children에 담겨있는 메뉴들도 routes 변수 요소로 담는 함수
```javascript
import type { RouteObject } from 'react-router-dom';
import { subRouteConfig } from './subRouteConfig';
import Error404 from '@/pages/auth/error/components/error404/Error404';
import Error403 from '@/pages/auth/error/components/error403/Error403';
import { MenuData } from '@/pages/common/types/menuTypes';

export const createMenuRoutes = (menus: MenuData[]): RouteObject[] => {
    const routes: RouteObject[] = [];

    const traverse = (menus: MenuData[]) => {
        menus.forEach((menu) => {
            if (menu.url && menu.menuKey) {
                if (menu.useYn !== 'Y') {
                    routes.push({
                        path: menu.url === '/' ? '' : menu.url,
                        element: <Error403 />,
                    });
                } else {
                    routes.push({
                        path: menu.url === '/' ? '' : menu.url,
                        children: subRouteConfig[menu.menuKey] ?? [
                            { index: true, element: <Error404 /> },
                        ],
                    });
                }
            }

            if (menu.children && menu.children.length > 0) {
                traverse(menu.children);
            }
        });
    };

    traverse(menus);
    return routes;
};
```

### BaseRoutes.tsx
- `RouterProvider` 로 router 설정
```javascript
import { Suspense, useMemo } from 'react';
import { createBrowserRouter, RouterProvider } from 'react-router-dom';
import { Navigate } from 'react-router-dom';
import useMenuStore from '@/stores/useMenuStore';
import { createMenuRoutes } from '@/navigation/routes/createMenuRoutes';
import Layout from '@/components/layout/layout';
import PlatformLayout from '@/components/layout/platformLayout';
import ComingSoon from '@/pages/auth/error/components/comingSoon/ComingSoon';
import Error404 from '@/pages/auth/error/components/error404/Error404';
import Error500 from '@/pages/auth/error/components/error500/Error500';
import Error403 from '@/pages/auth/error/components/error403/Error403';
import LoginPage from '@/pages/auth/login/LoginPage';
import HomePage from '@/pages/client/home/HomePage';

const BaseRoutes = () => {
    const userMenu = useMenuStore((s) => s.userMenu);
    const hasHydrated = useMenuStore.persist.hasHydrated();

    const menuRoutes = useMemo(() => createMenuRoutes(userMenu), [userMenu]);

    const router = useMemo(
        () =>
            createBrowserRouter([
                {
                    path: '/',
                    element: <Layout />,
                    children: [
                        {
                            element: <PlatformLayout />,
                            children: [{ index: true, element: <HomePage /> }, ...menuRoutes],
                        },
                        { path: 'coming-soon', element: <ComingSoon /> },
                        { path: 'error403', element: <Error403 /> },
                        { path: 'error500', element: <Error500 /> },
                        { path: '*', element: <Error404 /> },
                    ],
                },
                { path: '/login', element: <LoginPage /> },
                { path: '*', element: <Navigate to="/login" replace /> },
            ]),
        [menuRoutes],
    );

    if (!hasHydrated) {
        return <div>loading...</div>;
    }

    return (
        <Suspense fallback={<div>loading...</div>}>
            <RouterProvider router={router} key={`router-${userMenu.length}`} />
        </Suspense>
    );
};

export default BaseRoutes;
```
